PSU-Crypt

Description:

PSU-CRYPT is a C++ program that runs a Feistel cipher block encryption algorithm based on Twofish by Bruce Schneier, John Kelsey, Doug Whiting, David Wagner, Chris Hall, and SKIPJACK by the NSA - on a 64-bit block with a 80-bit key.

Responsibilities:
  • Implement an encryption mode and the encryption function
  • Implement a decryption mode and the decryption function
  • Add different key and plaintext/ ciphertext input options with different input mode for plaintext when in decryption mode (formats: binary string, hex string, ASCII string)
  • Create an encryption and decryption subkey generator, and the key scheduler
  • Write the F-function and G-function for the encrypt/ decrypt process
  • Add or remove ANSI X9.23 Padding from decrypted ciphertext and plaintext to be encrypted
