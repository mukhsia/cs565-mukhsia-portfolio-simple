PSU-PUBKEYCRYPT

Description:

PSU-PUBKEYCRYPT is a C++ program for a public key encryption scheme. The system must consist of the following three components: A 32-bit public & private key generator, An Ascii plaintext to ciphertext encryption sub-program, A ciphertext to Ascii plaintext decryption sub-program.

Generator:

Prompt the user for a random number with which to seed a random number generator. Compute e2 = g^d mod p. Output a public-key p g e2 and the corresponding private key p g d. These keys should be output into two files: pubkey.txt and prikey.txt. The format of these files should be: the first line of the file contains the integer key values and they are only separated by a space, as: p g d.

Encryption:

As input, the program take as plaintext a standard text file (call the file ptext.txt) and a public-key p g e2 contained in the file pubkey.txt. A value k is randomly picked from {0,..., p-1}, and is used to compute the following C1 = g∙k mod p C2 = e2∙k∙m mod p Output the ciphertext as pairs of integers C1 and C2 to a text file called ctext.txt. So if the message M = m1 m2 m3… mn then each mi will give a pair C1 and C2. For the graders convenience you can also print this to the screen.

Decryption:

As input, the program take as ciphertext a standard text file (ctext.txt) containing integer pairs C1 and C2 and a private-key p g d contained in the file prikey.txt. Output the decryption of the ciphertext into a file called dtext.txt.

Responsibilities:
  • Implement a square-and-multiply function for the mod-math computation
  • Implement a Miller-Rabin primality test function to perform primality testing on potential prime for the key generation process
  • Work out the computation needed for decryption
  • Include a safe modular multiplication and modular exponentiation functions to prevent 32-bit overflow
  • Develop the key generation, encryption, and decryption functions and the main 'wrapper' program
